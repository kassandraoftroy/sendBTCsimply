Installation Guide.

We are going to walk through the set up of our offline signing script from a Shell/Terminal. If you are unfamiliar with the command line - don't worry. We won't be doing anything fancy.

1. Open a Shell Session

      If you are on a mac you can type "Terminal" into the Spotlight.
      On windows the application is called "Command Prompt"
      when opened it should look something like this:

2. Get pip (python package installer)

      In the terminal type "sudo easy_install pip" and press enter

              $ sudo easy_install pip   

      you will be prompted for your password, referring to the device's User Password
      (if you have a password to sign in to your account when you first boot up your computer -- that's the one)
      Type your password and press enter.
      (you know it worked if there's a lot of activity and at the end it says something like "succefully installed pip...")
      having issues with this step? look here, here and here.

3. Download btcsignature package with pip

      In the terminal type "pip install btcsignature" and press enter

              $ pip install btcsignature   

      There should be a lot of activity in the terminal again (downloading dependencies)
      It should finish with "Succesfully installed btcsignature 0.11"
      *(if you got a "Permission Denied" Error try "$ sudo pip install btcsignature")*

4. You are ready to sign transactions from the command line

      Whenever you want to use the transaction signing tool just open a Shell/Terminal session.
      To run the signing script just type and enter "sign-offline"

              $ sign-offline   

      The script will then run (only if you are offline)
      When it prompts you: input your private key and press enter
      When it prompts you again: input your Unsigned Transaction Data and press enter
      You're done!
